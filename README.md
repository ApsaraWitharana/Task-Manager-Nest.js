# üìù Task Manager - CRUD API using NestJS

This is a simple **CRUD (Create, Read, Update, Delete)** Task Manager API built with [NestJS](https://nestjs.com/). It allows you to manage tasks using a RESTful API with in-memory storage.

---

## ‚úÖ Features

- Create a task
- Get all tasks
- Get a task by ID
- Update task status
- Delete a task
- In-memory data storage (no database required)
- Built with NestJS and TypeScript

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
